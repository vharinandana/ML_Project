# Bank Customer Churn Prediction

This project predicts whether a customer will churn using Machine Learning techniques.

## Features
- Interactive Streamlit Web Application
- Pre-trained Machine Learning Model
- Data preprocessing and feature encoding
- Real-time prediction

##  Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit

##  Dataset
- Bank Customer Churn Dataset

##  Live Demo
https://mlproject-5zu4cc9gnze4kbi59wzfnq.streamlit.app/

##  Project Structure
- app.py → Streamlit app
- project.ipynb → Model training
- *.pkl → Saved model & encoders
- requirements.txt → Dependencies

##  How to Run Locally
1. Clone the repository:
git clone [https://github.com/vharinandana/ML_Project.git](https://github.com/vharinandana/ML_Project.git)

2. Navigate to the directory:
cd ML_Project

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
streamlit run app.py
